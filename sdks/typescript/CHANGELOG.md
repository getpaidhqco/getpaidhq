## [0.1.0] - 2024-07-01

### Added

- Initial release of the GetPaidHQ TypeScript SDK
- Core client with automatic retries and error handling
- Resource-based API for subscriptions, customers, invoices, payments, and usage
- Comprehensive TypeScript typings for all API resources
- Examples for common use cases
- Documentation in README.md
